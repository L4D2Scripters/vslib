VSLib
=====

A simple, powerful VScript library for L4D2.
This library will make your modding so much easier. Create superior mutations, interactive maps, and much more.


Installing
==========

[Download the library](https://github.com/L4D2Scripters/vslib/archive/master.zip) and just extract the contents to left4dead2/scripts/vscripts/

For API documentation (e.g. Doxygen), tutorials, and links, see here:
http://l4d2scripters.github.io/vslib
